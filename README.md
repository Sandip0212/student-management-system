# 🎓 Student Management System

A Java Spring Boot REST API project built to manage student records. This project is being developed step by step as part of my Java Backend Developer learning journey.

## 🚀 Tech Stack

- Java 25
- Spring Boot 3.5
- Spring Data JPA
- Hibernate
- MySQL 8
- Maven
- Eclipse IDE
- Git & GitHub

## 📂 Project Structure

```
src
 └── main
```
